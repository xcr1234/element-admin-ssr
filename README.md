**开箱即用**

# 运行

1.安装依赖，执行 `yarn install` 或者 `npm install`

2.执行`npm run dev`

3.浏览器打开 localhost:3000

# 生产发布

1.执行`npm run build`

2.编译后的所有文件在dist/下，拷贝到生产目录

3.把node_modules拷贝到生产目录，或者去生产目录执行`npm install`

4.执行`npm run start`
